# Fashion AI 실행 방법

## 방법 1: 배치 파일 실행 (추천)
1. **`실행.bat`** 파일을 더블클릭
2. 브라우저가 자동으로 열립니다
3. 종료하려면 실행 창을 닫으세요

## 방법 2: PowerShell 스크립트
1. **`실행.ps1`** 파일을 우클릭 → "PowerShell에서 실행"
2. 브라우저가 자동으로 열립니다
3. 종료하려면 Ctrl+C

## 방법 3: 수동 실행
1. 터미널/명령 프롬프트를 엽니다
2. 이 폴더로 이동합니다
3. 다음 명령어를 입력:
   ```bash
   python -m http.server 8000
   ```
4. 브라우저에서 `http://localhost:8000` 접속

## 주의사항
- **index.html을 직접 더블클릭하면 작동하지 않습니다!**
- 반드시 로컬 서버를 통해 실행해야 합니다.
- Python이 설치되어 있어야 합니다.

## 문제 해결
- "python을 찾을 수 없습니다" 오류가 나면:
  - Python을 설치하거나
  - `python3 -m http.server 8000` 시도

- 포트 8000이 이미 사용 중이면:
  - 다른 포트 사용: `python -m http.server 8080`
  - 브라우저에서 `http://localhost:8080` 접속
